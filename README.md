# Auditoria de Segurança e Compliance – Botium Toys

## 📌 Problema
A empresa Botium Toys precisava avaliar seus controles de segurança e conformidade para identificar vulnerabilidades e riscos operacionais.

## 🔍 Análise
Foi realizada uma análise baseada em:
- Controles de segurança (Firewall, IDS, Backups, etc.)
- Práticas de conformidade (PCI DSS, GDPR, SOC)
- Relatório de avaliação de riscos

Cada controle foi verificado para identificar se estava implementado ou não.

## ✅ Decisão (YES/NO)
- Controles ausentes ou não documentados foram marcados como **NO**
- Controles existentes e comprovados foram marcados como **YES**

Essa abordagem garante uma visão realista da postura de segurança da empresa.

## ⚠️ Principais Riscos Identificados
- Falta de criptografia em dados sensíveis
- Políticas de senha fracas
- Ausência de plano de resposta a incidentes
- Controle de acesso inadequado

## 🛡️ Recomendações
- Implementar políticas de senha fortes
- Adotar criptografia para dados sensíveis
- Criar plano de resposta a incidentes
- Aplicar princípio de privilégio mínimo
- Implantar backups automáticos e testados

## 🚀 Resultado
A análise permitiu identificar falhas críticas e propor melhorias para fortalecer a segurança e a conformidade da empresa.
